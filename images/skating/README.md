Place ice skating gallery photos in this folder.

For the group photo shown with the Ice Skating Adventure request, use one of
these filenames:

- ice-skating-adventure.jpg
- ice-skating-adventure.jpeg
- ice-skating-adventure.png
- ice-skating-adventure.webp

Best option: save it as `images/skating/ice-skating-adventure.jpg`.

Recommended filenames:

- skating-01.jpg
- skating-02.jpg
- skating-03.jpg
- skating-04.jpg
- skating-05.jpg
- skating-06.jpg
- skating-07.jpg
- skating-08.jpg

The page checks `skating-01` through `skating-24` automatically in `.jpg`,
`.jpeg`, `.png`, and `.webp` formats. It also checks the special
`ice-skating-adventure` filename and captions it as "Ice Skating Adventure".
Add more names to the `skatingImages` list in `script.js` if you have
additional photos or different filenames.
